# Information Security Fundamentals (ISFS)

A comprehensive documentation repository covering the core pillars, frameworks, and control sets required for Information Security management.

## 📂 Repository Contents

### Core Concepts & Frameworks
*   **[CIA.md](./CIA.md)**: Deep dive into the Confidentiality, Integrity, and Availability triad.
*   **[risk-management.md](./risk-management.md)**: Strategies for identifying, evaluating, and mitigating security risks.
*   **[ISO27001_vs_27002_Comparison.md](./ISO27001_vs_27002_Comparison.md)**: A side-by-side analysis of the requirements (27001) vs. the code of practice (27002).
*   **[Incident-Management-Cycle.md](./Incident-Management-Cycle.md)**: Procedures for detecting, responding to, and recovering from security incidents.

### Security Control Domains
Documentation regarding the implementation and management of specific control types:
*   **[Organizational-Controls.md](./Organizational-Controls.md)**: Governance, policies, and procedural standards.
*   **[People-Controls.md](./People-Controls.md)**: Security awareness, hiring practices, and user behavior.
*   **[Physical-Controls.md](./Physical-Controls.md)**: Securing facilities, hardware, and environmental safety.
*   **[Technological-Controls.md](./Technological-Controls.md)**: Logic-based protections like encryption, MFA, and firewalls.
