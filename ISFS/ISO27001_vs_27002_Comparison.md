# Difference Between ISO/IEC 27001 and ISO/IEC 27002

## ISO/IEC 27001
- An **international standard** for Information Security Management Systems (ISMS)
- Defines **requirements** for establishing, implementing, maintaining, and improving ISMS
- **Certifiable** (organizations can get officially audited and certified)
- Focuses on:
  - Risk management
  - Policies and procedures
  - Continuous improvement

👉 **In short:** Defines *what* needs to be done.

---

## ISO/IEC 27002
- A **guideline** (not a certifiable standard)
- Provides **best practices** for implementing security controls
- Expands on controls referenced in ISO 27001
- Helps organizations apply security measures effectively

👉 **In short:** Explains *how* to do it.

---

## Key Differences

| Aspect        | ISO/IEC 27001              | ISO/IEC 27002              |
|--------------|--------------------------|----------------------------|
| Type         | Formal standard           | Guideline                  |
| Certification| Yes                       | No                         |
| Purpose      | Build and manage ISMS     | Implement security controls|
| Focus        | Requirements              | Best practices             |

---

## Relationship
- ISO/IEC 27001 **requires** security controls
- ISO/IEC 27002 **explains** those controls

➡️ They **complement each other** and are often used together.
