#  CISM Study & Resources Repository

## 📌 Overview
This repository contains study notes, and resources for the **Certified Information Security Manager (CISM)** certification by ISACA.

CISM is a globally recognized certification focused on **information security governance, risk management, and incident management**.

---

## 🎯 Objectives
- Provide structured notes for all CISM domains  
- Share study guides, summaries, and cheat sheets  
- Help track learning progress  
- Support exam preparation

---

## CISM Domains

### 1. Information Security Governance
- Aligning security strategy with business goals  
- Policy development and management  

### 2. Information Risk Management
- Risk identification, assessment, and mitigation  
- Risk reporting and communication  

### 3. Information Security Program Development & Management
- Building and managing security programs  
- Resource and lifecycle management  

### 4. Information Security Incident Management
- Incident response planning  
- Detection, response, and recovery
