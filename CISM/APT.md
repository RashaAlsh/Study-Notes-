# CISM Threat Intelligence: APT Lifecycle

| Stage | Action | ISM Focus |
| :--- | :--- | :--- |
| **1. Reconnaissance** | Gathering intel on employees and tech stacks. | Security Awareness / OSINT reduction. |
| **2. Initial Intrusion** | Phishing, 0-day exploits, or supply chain compromise. | Defense in Depth / MFA / EDR. |
| **3. Command & Control (C2)** | Establishing a "beacon" to the attacker's server. | Egress Filtering / Network Analytics. |
| **4. Lateral Movement** | Moving from a low-privilege PC to a Domain Admin. | PAM / Network Segmentation. |
| **5. Exfiltration** | Packaging and sneaking data out of the network. | DLP / Traffic Baselines. |
| **6. Persistence** | Creating "backdoors" to return if discovered. | Root Cause Analysis (RCA). |
