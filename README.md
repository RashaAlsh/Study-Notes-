## NIS2
- EU Directive on **Network and Information Systems Security**
- Covers cybersecurity requirements for critical infrastructure and essential services
- Key objectives:
  - Risk management measures
  - Reporting of incidents
  - Cooperation between EU member states
- Applicable to organizations in energy, transport, banking, health, and digital services

## AZ-500 (Microsoft Azure Security Technologies)
- Certification for **Azure security engineers**
- Key skills covered:
  - Manage **identity and access** in Azure (IAM, Azure AD, RBAC)
  - Implement **platform protection** (VM, networking, Azure Security Center)
  - Manage **security operations** (monitoring, threat detection, response)
  - Secure **data and applications** in Azure (Key Vault, encryption, secrets)
  - Manage **governance and compliance** in Azure
- Focus on hands-on Azure security best practices
